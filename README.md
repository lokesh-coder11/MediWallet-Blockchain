# 🏥 MediWallet using Ethereum Blockchain
Secure Electronic Health Record System using Ethereum Blockchain

## 📌 Overview

MediWallet is a decentralized healthcare record management system built using blockchain technology. It allows patients to securely store and manage their medical records while giving them full control over who can access their data.

The system uses Ethereum smart contracts for access control and IPFS for secure storage, ensuring data integrity, transparency, and privacy.

---

## 🚀 Features

* 🔐 Secure storage of medical records
* 👤 Patient-controlled access (grant/revoke permissions)
* 🧑‍⚕️ Doctor access request system
* 📜 Tamper-proof records using blockchain
* 🔗 Integration with MetaMask wallet
* ☁️ Decentralized file storage using IPFS

---

## 🛠 Tech Stack

* **Frontend:** React
* **Blockchain:** Ethereum
* **Smart Contracts:** Solidity
* **Web3 Integration:** ethers.js / web3.js
* **Storage:** IPFS
* **Wallet:** MetaMask

---

## 🧠 System Architecture

React (Frontend) → Web3 (ethers.js) → Smart Contract (Solidity) → IPFS (Storage)

---

## ⚙️ How It Works

1. User logs in using MetaMask wallet
2. Patient uploads medical records to IPFS
3. IPFS returns a unique hash
4. Hash is stored on Ethereum blockchain via smart contract
5. Doctor requests access to records
6. Patient grants/revokes permission using smart contract
7. Authorized users can view records using IPFS hash

## ▶️ How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/your-username/MediWallet-Blockchain.git
cd MediWallet-Blockchain
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the frontend

```bash
npm start
```

### 4. Deploy Smart Contracts

* Use Ganache or Ethereum testnet
* Deploy using Truffle / Hardhat
